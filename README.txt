# Result_Automation
This is a python project which helps in calculating the results and percentages of students.
Ideally, this will be for school teachers, who want to calculate percentages and CGPA but are fed up with the calculator usage.
How to use:
1. Either run the python file or the exe file.
2. Enter number of subjects.
3. Enter marks of subjects.
4. That's all!!

Note: Marks are to be out of 100.
